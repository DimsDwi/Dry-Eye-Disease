# Dry-Eye-Disease
Dry Eye Disease adalah kondisi ketika mata tidak menghasilkan air mata yang cukup sehingga menyebabkan iritasi, perih, dan kelelahan mata. Aplikasi ini menggunakan Artificial Intelligence dan algoritma Support Vector Machine (SVM) untuk menganalisis risiko mata kering berdasarkan gaya hidup dan gejala pengguna secara cepat dan interaktif.
